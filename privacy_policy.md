# 一球速報 プライバシーポリシー

最終更新日: 2026年6月27日

岩井賢太郎（以下「開発者」）は、iOSアプリ「一球速報」（以下「本アプリ」）に
おけるユーザーの個人情報の取り扱いについて、以下のとおりプライバシーポリシー
（以下「本ポリシー」）を定めます。

---

## 1. 取得する情報

本アプリは以下の情報を取得し、利用します。

### 1-1. アカウント情報
- Google アカウントによるサインイン時に、Google から提供される
  メールアドレス、表示名、Google アカウントの一意な識別子（Firebase UID）

### 1-2. 利用情報
- ユーザーが本アプリ内で登録するチーム情報、選手情報、試合記録、打順、
  投球記録、得点、観戦コード等

### 1-3. デバイス情報
- プッシュ通知配信のための APNs デバイストークン
- アプリのクラッシュ・エラー情報（個人を特定する情報は含みません）

### 1-4. 画像情報
- メンバー表 OCR 機能を利用する際に、ユーザーが選択・撮影した画像
  （後述する Gemini API による解析後、原則として保存しません）

---

## 2. 情報の利用目的

取得した情報は、以下の目的で利用します。

1. 本アプリの機能提供（試合記録・観戦・通知等）
2. ユーザーが所属するチームのメンバー間での情報共有
3. プッシュ通知の配信
4. メンバー表 OCR による選手情報の自動入力
5. 不具合の調査・改善
6. お問い合わせへの対応

---

## 3. 第三者提供・委託先

本アプリは、サービス提供のために以下の外部サービスを利用しています。
情報はこれらのサービスに送信されますが、それぞれの利用規約・プライバシー
ポリシーに従って取り扱われます。

| サービス | 提供元 | 用途 |
|---|---|---|
| Firebase Authentication | Google LLC | ログイン認証 |
| Firebase Cloud Firestore | Google LLC | 試合状態のリアルタイム同期 |
| Supabase (PostgreSQL) | Supabase Inc. | チーム・試合・選手情報の保存 |
| Google Cloud Run | Google LLC | バックエンドAPIのホスティング |
| Apple Push Notification Service | Apple Inc. | プッシュ通知配信 |
| Gemini API | Google LLC | メンバー表画像のOCR解析 |
| Sentry | Functional Software, Inc. | エラー情報の記録 |

ユーザーが本アプリで登録した情報は、同じチームに所属する他のメンバーに
共有されることがあります（例: 試合記録、選手情報、打順）。

上記以外の第三者に、ユーザーの個人情報を提供することはありません。
ただし、法令に基づく開示請求があった場合はこの限りではありません。

---

## 4. データの保管期間

- ユーザーが本アプリを継続して利用している間、登録情報は保持されます。
- アカウント削除のリクエストがあった場合、開発者は合理的な期間内に
  対象データを削除します。
- 法令により保管が義務付けられている場合は、その期間保管します。

---

## 5. データの削除・退会

アカウント削除を希望する場合は、後述のお問い合わせ先までご連絡ください。
チームから退会する操作はアプリ内から可能です。

---

## 6. 子どもの個人情報

本アプリは少年野球の試合記録を主な目的としており、未成年者の選手情報
（氏名、学年、背番号等）が登録されることがあります。これらは保護者・
チーム関係者の責任のもとで登録・管理されることを前提としています。

本アプリは13歳未満のユーザー自身による登録は想定していません。
13歳未満の選手情報は、保護者または所属チームの管理者が責任を持って
登録してください。

---

## 7. セキュリティ

本アプリは、通信時に HTTPS による暗号化を行い、データの保護に努めています。
ただし、インターネット上の通信について100%の安全性を保証するものでは
ありません。

---

## 8. 広告について

本アプリは、サービス提供のため、Google LLC が提供する広告配信サービス
（Google AdMob）を利用しています。広告はホーム画面・速報観戦画面・公開観戦画面
に表示されます（試合記録画面には表示されません）。

広告サービスは、ユーザーに関連性の高い広告を表示するため、広告識別子（IDFA）
や Cookie 等の情報を利用することがあります。これらの情報の収集・利用方法に
ついては、Google のプライバシーポリシーをご確認ください。

- [Google プライバシーポリシー](https://policies.google.com/privacy)
- [Google の広告に関するポリシー](https://policies.google.com/technologies/ads)

### ATT（App Tracking Transparency）について

iOS では、Apple の App Tracking Transparency の仕組みにより、アプリ起動時に
トラッキングの許可をお伺いします。許可いただいた場合のみ、広告識別子（IDFA）
が広告のパーソナライズに利用されます。許可しなかった場合も、引き続き広告は
表示されますが、パーソナライズされない広告となります。

### 広告トラッキングのオプトアウト

広告トラッキングを後から変更したい場合は、iPhone の「設定」→「プライバシーと
セキュリティ」→「トラッキング」から、本アプリのトラッキング許可を変更できます。
また、Google の広告設定から、Google 広告のパーソナライズを無効にすることもできます。

- [Google 広告設定（オプトアウト）](https://adssettings.google.com/)
