一球速報 プライバシーポリシー

最終更新日: 2026年5月19日

岩井賢太郎（以下「開発者」）は、iOSアプリ「一球速報」（以下「本アプリ」）におけるユーザーの個人情報の取り扱いについて、以下のとおりプライバシーポリシー（以下「本ポリシー」）を定めます。

1. 取得する情報

本アプリは以下の情報を取得し、利用します。

1-1. アカウント情報

  • Google アカウントによるサインイン時に、Google から提供されるメールアドレス、表示名、Google アカウントの一意な識別子（Firebase UID）

1-2. 利用情報

  • ユーザーが本アプリ内で登録するチーム情報、選手情報、試合記録、打順、投球記録、得点、観戦コード等

1-3. デバイス情報

  • プッシュ通知配信のための APNs デバイストークン
  • アプリのクラッシュ・エラー情報（個人を特定する情報は含みません）

1-4. 画像情報

  • メンバー表 OCR 機能を利用する際に、ユーザーが選択・撮影した画像（後述する Gemini API による解析後、原則として保存しません）

2. 情報の利用目的

取得した情報は、以下の目的で利用します。

  1. 本アプリの機能提供（試合記録・観戦・通知等）
  2. ユーザーが所属するチームのメンバー間での情報共有
  3. プッシュ通知の配信
  4. メンバー表 OCR による選手情報の自動入力
  5. 不具合の調査・改善
  6. お問い合わせへの対応

3. 第三者提供・委託先

本アプリは、サービス提供のために以下の外部サービスを利用しています。情報はこれらのサービスに送信されますが、それぞれの利用規約・プライバシーポリシーに従って取り扱われます。

サービス提供元用途
Firebase AuthenticationGoogle LLCログイン認証
Firebase Cloud FirestoreGoogle LLC試合状態のリアルタイム同期
Supabase (PostgreSQL)Supabase Inc.チーム・試合・選手情報の保存
Google Cloud RunGoogle LLCバックエンドAPIのホスティング
Apple Push Notification ServiceApple Inc.プッシュ通知配信
Gemini APIGoogle LLCメンバー表画像のOCR解析
SentryFunctional Software, Inc.エラー情報の記録

ユーザーが本アプリで登録した情報は、同じチームに所属する他のメンバーに共有されることがあります（例: 試合記録、選手情報、打順）。

上記以外の第三者に、ユーザーの個人情報を提供することはありません。ただし、法令に基づく開示請求があった場合はこの限りではありません。

4. データの保管期間

  • ユーザーが本アプリを継続して利用している間、登録情報は保持されます。
  • アカウント削除のリクエストがあった場合、開発者は合理的な期間内に対象データを削除します。
  • 法令により保管が義務付けられている場合は、その期間保管します。

5. データの削除・退会

アカウント削除を希望する場合は、後述のお問い合わせ先までご連絡ください。チームから退会する操作はアプリ内から可能です。

6. 子どもの個人情報

本アプリは少年野球の試合記録を主な目的としており、未成年者の選手情報（氏名、学年、背番号等）が登録されることがあります。これらは保護者・チーム関係者の責任のもとで登録・管理されることを前提としています。

本アプリは13歳未満のユーザー自身による登録は想定していません。13歳未満の選手情報は、保護者または所属チームの管理者が責任を持って登録してください。

7. セキュリティ

本アプリは、通信時に HTTPS による暗号化を行い、データの保護に努めています。ただし、インターネット上の通信について100%の安全性を保証するものではありません。

8. プライバシーポリシーの変更

本ポリシーは、必要に応じて変更されることがあります。変更があった場合、本ページに掲載することで通知します。
